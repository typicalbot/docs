# Mute

Mute a member in the server.

## Permission

Requires 'Manage Roles' and 'Manage Channels'

## Usage

 - `$mute <user>`
 - `$mute <user> [reason]`
 - `$mute <user> [duration]`
 - `$mute <user> [duration] [reason]`
 - `$mute deny here`
 - `$mute deny [channel]`

## Examples

 - `$mute deny here`
 - `$mute @Xenotater Spam`
 - `$mute @Xenotater 2d`
