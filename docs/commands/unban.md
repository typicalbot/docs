# Unban

Unban a member from the server.

## Permission

Requires 'Ban Members'

## Usage

 - `$unban <user>`
 - `$unban <user> [reason]`

## Examples

 - `$unban 187980183569956864`
 - `$unban 187980183569956864 Agreed to stop shamelessly promoting his YouTube channel`
