# Conf

View or customize your servers setting and enable/disable specific features.

## Usage

 - `$conf list`
 - `$conf clear`
 - `$conf <setting>`
 - `$conf <setting> <value>`
 - `$conf <setting> add <value>`
 - `$conf <setting> remove <value>`
